## Overview

The Performance module supports structured review cycles with configurable templates. It covers the full feedback loop: goal setting at the start of a period, self-assessments during the cycle, and calibrated manager ratings at close.

## What a review cycle includes

| Component | Description |
|---|---|
| **Objectives** | Goals set at the start of the period, with status tracking throughout |
| **Self-assessment** | Employee-authored reflection against their objectives |
| **Manager review** | Manager evaluation of the employee's performance against each objective |
| **Final rating** | Calibrated score recorded permanently in the employee's profile |

## Creating a review cycle

<Steps>
  <Step>
    Navigate to **HRM → Performance**.
  </Step>
  <Step>
    Click **New cycle**.
  </Step>
  <Step>
    Set the **cycle name**, **period** (start and end dates), and the **employees in scope**.
  </Step>
  <Step>
    Choose a **review template** — your administrator configures available templates in HRM settings.
  </Step>
  <Step>
    Click **Launch cycle**. Employees in scope receive a notification to complete their self-assessments.
  </Step>
</Steps>

## Completing a review

Employees and managers complete their sections independently. Progress is visible in the **cycle dashboard** so administrators can track completion rates and send reminders.

Once both the self-assessment and manager review are submitted, the manager **finalises the rating** — this commits the calibrated score to the employee's profile.

<Callout type="info" title="Finalising is audited">
  Finalising a rating writes a permanent audit event linked to the cycle, the employee, and the rating value. Ratings can only be revised by creating a new review cycle.
</Callout>

## Objectives

Objectives can be set at the **individual** or **team** level. Each objective carries a status that updates throughout the cycle:

- **Not started** — committed but work has not begun
- **In progress** — actively being worked on
- **Completed** — achieved; ready for manager evaluation
- **Cancelled** — dropped from the cycle scope
