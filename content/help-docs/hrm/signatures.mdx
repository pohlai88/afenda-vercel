# HRM signatures

Afenda records in-app signature ceremonies with a tamper-evident JSON envelope stored in the document vault.

## Workbench

HR users create a ceremony from the employee contract panel or the **Signatures** workbench, attach the source PDF, define parties, and send invitations.

## Portal

Employees sign at **My signatures** using their portal session. Each party receives an opaque token URL.
